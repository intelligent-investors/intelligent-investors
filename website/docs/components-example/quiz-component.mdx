# Quiz Component

The `Quiz` component allows you to render a quiz within your document.

## API

### `Quiz`

| Property     | Description                                      | Type   | Default |
|--------------|--------------------------------------------------|--------|---------|
| description  | The description of the quiz.                     | String |         |
| questions    | An array of question objects for the quiz.       | `Array<QuizQuestion>`  |       |

### `QuizQuestion`

A `QuizQuestion` object should have the following properties:

| Property     | Description                                      | Type   | Default |
|--------------|--------------------------------------------------|--------|---------|
| notes        | Any additional notes about the question.         | String |       |
| question     | The question text.                               | String |       |
| answers       | An array of possible answers for the question.  | `Array<QuizAnswerOption>` |      |
| explanation  | The explanation for all answer options (supports math equations and multiple lines).  | String |       |

### `QuizAnswerOption`

An `QuizAnswerOption` object should have the following properties:

| Property     | Description                                      | Type   | Default |
|--------------|--------------------------------------------------|--------|---------|
| value        | The text of the answer option.                   | String |       |
| correct      | Whether this answer is correct (`true` or `false`). | Boolean |   |
